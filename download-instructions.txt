NILO WEBSITE - DOWNLOAD INSTRUCTIONS
===================================

ESSENTIAL FILES TO DOWNLOAD:
1. package.json
2. vite.config.ts
3. tailwind.config.ts
4. tsconfig.json
5. All files in 'client' folder
6. All files in 'server' folder
7. All files in 'shared' folder

HOW TO DOWNLOAD INDIVIDUAL FILES:
1. Click on each file in the left sidebar
2. Copy the content (Ctrl+A, then Ctrl+C)
3. Create new files on your laptop with the same names
4. Paste the content

SETUP ON YOUR LAPTOP:
1. Install Node.js from nodejs.org
2. Create a new folder called 'nilo-website'
3. Put all downloaded files in their correct folders
4. Open terminal in the project folder
5. Run: npm install
6. Create .env file with: OPENAI_API_KEY=your_key_here
7. Run: npm run dev

Your website will be available at http://localhost:5000
